# Roadmap

## Phase 1: MVP

- Landing page
- Authentication
- User profile
- Document upload
- Basic Form 16 extraction
- Manual income and deduction entry
- Old/new regime comparison
- ITR recommendation
- PDF summary export

## Phase 2: Intelligence

- AIS and Form 26AS parsing
- Broker statement analysis
- Capital gains engine
- Deduction finder
- Validation engine
- AI tax assistant with RAG

## Phase 3: SaaS Scale

- Admin dashboard
- Notifications
- Celery processing pipeline
- Audit logs
- Advanced security
- Subscription billing

## Future

- Direct Income Tax e-Filing portal integration subject to official APIs and compliance
- UPI payment for tax dues
- GST return assistance
- CA collaboration workspace
- Tax planning throughout the year
- Portfolio integrations
- AI audit risk scoring
- Notice explanation
- Mobile app
